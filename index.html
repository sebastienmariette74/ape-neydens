Les membres du bureau sont élus lors de la réunion de la rentrée scolaire. Participent à cette élection tous les parents présents.

Cette réunion permet également de répartir les missions entre tous les membres bénéboles de l'association dans le but de préparer les différents évènements de l'année.

Parmi nos bénévoles, nous comptons : Albane, Gilles, Fred L., Christophe D., Christophe A., Elisa, Damien, Jean-Louis, Mélanie, Marine, Odile, Jérôme, Matthieu, Sabrina, Cathy, Joffrey.

Nous ne serons jamais assez nombreux alors si vous voulez nous rejoindre, n'hésitez pas à nous contacter. On peut vous garantir d'excellents moments de joie et de convivialité. L'association est ouverte à tous !!!

Notre école est située dans l'agglomération de Neydens en Haute-Savoie. Pour l'année scolaire 2021-2022, 198 enfants y sont inscrits, répartis dans 8 classes :

  • PS / MS
  • PS / GS
  • MS / GS
  • GS / CP
  • CP / CE1
  • CE1 / CE2
  • CE2 / CM2
  • CM1 / CM2
